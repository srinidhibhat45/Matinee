Get Started in 3 Steps

Matinee operates with full local independence. Here is how to configure it in minutes.

1

Download & Install the APK

Click the download button to save build-latest.apk directly to your Android device.

Note: Since it is an offline APK distributed directly, you might need to enable "Allow installation from Unknown Sources" in your browser or file manager settings.

Download APK
2

Configure Your API Keys

Matinee operates client-side with complete privacy. To query movie metadata and generate personalized AI insights, you will need to add two keys in the app's settings panel:

A. Google Gemini API Key (For AI Taste Insights)

  1. Visit Google AI Studio.
  2. Sign in with your Google account.
  3. Click Create API Key and copy your key.

B. TMDb API Read Access Token (For Media Metadata)

  1. Sign up or log in to The Movie Database (TMDb).
  2. Go to your Account Settings → **API** tab.
  3. Request an API key (registering as a Developer for personal use).
  4. Copy the **API Read Access Token** (the long JWT key).

Both keys are stored securely on your device's local encrypted sandbox (Expo SecureStore) and never pass through any middleman or external servers.

3

Activate & Enjoy

Open Matinee, navigate to the Profile & Settings tab, and paste both keys in their respective fields. The app will immediately initialize your local candidate recommendations seeds based on your database rating history.
